# V2 战斗术语词典

游戏内战斗相关专有名词，结合游戏内描述与内核公式整理。

---

## 行动类型

### 重击
- **内核标识**：v1 — `"attack"` type 最后一段，由 STAGGER 事件识别；v2 — `"heavy_attack_hit"` trigger event
- **定义**：普通攻击连段的最后一段。仅主控干员可施放。命中时造成失衡并恢复技力
- **机制**：
  - 失衡值 = 技能描述中的重击恢复技力值（数值相等，但失衡不受增幅）
  - 恢复技力 = 技能描述中的值，受源石技艺强度增幅
  - 连携技触发条件 `on_heavy_attack` 基于 STAGGER 事件 + 来源为 attack 类型判定
- **注意**：不同角色重击的失衡值/恢复技力不同（管理员 18，秋栗 17 等），从各角色 skills.json 读取

## 角色特殊机制

### 源石结晶 (ENDMINISTRATOR)
- **内核标识**：`endmin_debuff`，载体 buff（敌方 debuff）
- **定义**：管理员连携技命中时附着在敌人身上的特殊 debuff。持续期间内，敌人被施加物理异常或破防时消耗结晶并造成额外伤害
- **来源**：连携技命中（skills.json `封印时间` 行控制持续时间：rank1-9 = 4s, M1-M2 = 4.5s, M3 = 5s）
- **消耗触发条件**：
  1. 敌人被施加任意物理异常（击飞/倒地/猛击/碎甲），任意角色均可触发
  2. 管理员终结技命中（DAMAGE_TICK + source = ultimate）
- **消耗时效果**（immediate + deferred 两阶段）：
  - **Immediate**：入队击碎结晶伤害（`击碎结晶伤害倍率`，rank3 = 213%）
  - **Immediate**：P1 返还 50 技力
  - **Deferred**（子树结算完毕后）：
    - 移除结晶 buff
    - 移除现实静滞 debuff
    - 天赋「本质瓦解」：管理员攻击力 +15/30%，15s，不可叠加
    - P2「权能映射」：队友获得本质瓦解一半的攻击力加成
- **关联天赋**：
  - **本质瓦解**（天赋 0，精英 1/2 解锁）：结晶消耗后 ATK +15/30%，15s
  - **现实静滞**（天赋 1，精英 2/3 解锁）：结晶存在期间敌人受到的物理伤害 +10/20%（脆弱乘区）
- **注意**：破防（break_apply）也触发消耗（Route 2.6），不仅限于四种物理异常

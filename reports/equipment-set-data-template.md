# 装备套装 3 件套效果数据模板

已实现: 点剑、动火用、脉冲式、潮涌
待填入: 以下 17 套

---

## 格式说明

```
套装名:
  常驻: <3件套常驻属性加成，如"源石技艺强度+30" 或 "物理伤害+20%">
  触发: <触发事件描述，如"施放战技后"、"施加物理异常后">
  效果: <触发后的效果，如"物理伤害+20%,10s" 或 "ATK+15%,15s,最多3层">
  ICD: <冷却秒数，如15s；无则填"无">
  备注: <特殊条件等>
```

- **常驻**: 装备3件套后永久生效的属性加成（已通过 timelineStore delta 注入 stats）
- **触发+效果**: 满足条件后临时生效的 buff（需要在 definitions.ts 实现）

## 已实现套装参考

### 点剑（7 件）
- 常驻: 物理伤害+28%
- 触发: 施加物理异常后
- 效果: 造成 ATK×250% 物理伤害 + 10 失衡
- ICD: 15s

### 动火用（8 件）
- 常驻: 源石技艺强度+30
- 触发: 对敌人施加燃烧/腐蚀后
- 效果: 燃烧→灼热附着+50%；腐蚀→自然附着+50%
- ICD: 无

### 脉冲式（5 件）
- 常驻: 源石技艺强度+30
- 触发: 对敌人施加导电/冻结后
- 效果: 导电→电磁附着+50%；冻结→寒冷附着+50%
- ICD: 无

### 潮涌（4 件）
- 常驻: 法术伤害+28%
- 触发: 任意法术附着层数>=2时
- 效果: 法术伤害+35%, 15s
- ICD: 无
